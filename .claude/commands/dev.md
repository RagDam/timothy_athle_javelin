# Démarrer le serveur de développement

Lance le serveur Next.js sur http://localhost:3000

## Instructions

1. D'abord, tue tous les processus Node.js existants pour libérer les ports
2. Ensuite, lance `npm run dev` dans le répertoire du projet
3. Indique à l'utilisateur que le serveur est disponible sur http://localhost:3000

Utilise PowerShell pour exécuter ces commandes.
