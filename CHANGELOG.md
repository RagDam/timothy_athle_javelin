## [0.1.0] - 2024-12-23

### Ajouté
- Initialisation du projet Next.js 16 avec App Router
- Configuration TypeScript strict
- Configuration Tailwind CSS v4
- Configuration ESLint
- Création de CLAUDE.md (règles de développement)
- Création de SPECIFICATIONS.md (recueil des specs)
- Création de CHANGELOG.md (ce fichier)

### Décisions
- Architecture monorepo (pas de séparation back/front pour le MVP)
- Dark mode par défaut
- Contenu géré via Markdown
- 7 pages MVP: Accueil, À propos, Palmarès, Médias, Agenda, Découvrir, Contact
